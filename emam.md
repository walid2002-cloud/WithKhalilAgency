# Emam – With Khalil | Agence Digital Marketing & Video Production

Projet Next.js pour le site de l'agence **With Khalil**.

## Technologies

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**
- **Lucide React** (icônes)

## Lancement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Structure principale

- `src/app/page.tsx` – Page d'accueil
- `src/components/` – Composants React
  - `emam123.tsx` – Section **Notre Méthode** (4 étapes premium)
  - `HeroTVParallax.tsx` – Hero
  - `SansVsAvec.tsx` – Comparaison Sans vs Avec
  - `Portfolio.tsx` – Portfolio
  - `Contact.tsx` – Formulaire de contact
  - `Navbar.tsx` – Navigation

## Build

```bash
npm run build
npm start
```
